# 04 Users And Permissions

这一章的重点是把“会操作文件”推进到“知道谁能操作、为什么能操作、权限不够时该怎么排查”。

学完这一章后，你应该能做到：

- 理解用户、用户组、root 分别是什么
- 看懂 `ls -l` 里的权限位
- 知道普通文件和目录的权限含义有什么不同
- 能使用 `chmod`、`chown`、`chgrp` 做基础权限调整
- 知道什么时候该用 `sudo`，什么时候不该乱用

## 建议学习顺序

1. 先看 `notes.md`
2. 再练 `commands.md`
3. 然后做 `labs/01-permissions-basics.md`
4. 最后用 `questions.md` 自测

## 文件说明

- `notes.md`: 用户、组、权限模型和常见排查思路
- `commands.md`: 这一章重点命令和典型用法
- `questions.md`: 复盘题和参考答案
- `labs/01-permissions-basics.md`: 第一轮权限实验

## 本章建议重点关注

- “我是谁” 和 “这个文件属于谁” 是两件不同的事
- `rwx` 在文件和目录上的含义并不完全一样
- 为什么有时明明文件存在，却还是提示权限不够
- `chmod 644` 和 `chmod 755` 各自意味着什么
- 为什么 `sudo` 很方便，但不应该默认到处都用

## 本章完成标准

如果下面这些你都能做到，说明这一章学得比较稳：

- 能解释用户、用户组、root 的基本作用
- 能看懂 `-rw-r--r--`、`drwxr-xr-x` 这类权限表示
- 能独立使用 `whoami`、`id`、`groups`、`chmod`
- 能说清目录的 `x` 权限为什么很关键
- 遇到 `Permission denied` 时，知道先从用户、路径、权限三方面排查
