# 03 Filesystem And Text

这一章的重点是把“会操作文件”推进到“会读文件、找文件、筛文本”。

学完这一章后，你应该能做到：

- 对 Linux 文件系统有更清晰的整体认识
- 知道普通文件、目录、链接这些对象的基本区别
- 能查看文件内容的开头、结尾和长文本
- 能按名称查文件，按关键词筛文本
- 能做基础统计、排序和去重

## 建议学习顺序

1. 先看 `notes.md`
2. 再练 `commands.md`
3. 然后做 `labs/01-files-and-logs.md`
4. 最后用 `questions.md` 自测

## 文件说明

- `notes.md`: 文件系统和文本处理的核心概念
- `commands.md`: 这一章重点命令和典型用法
- `questions.md`: 复盘题和参考答案
- `labs/01-files-and-logs.md`: 第一轮文件与日志实验

## 本章建议重点关注

- 路径和文件系统是怎么组织在一起的
- `cat`、`less`、`head`、`tail` 分别适合什么场景
- `grep`、`find`、`wc`、`sort`、`uniq` 是怎么配合使用的
- “查文件”和“查文件内容”有什么区别
- 为什么日志排查经常离不开 `tail` 和 `grep`

## 本章完成标准

如果下面这些你都能做到，说明这一章学得比较稳：

- 能解释目录、普通文件、链接的大致区别
- 能独立使用 `cat`、`less`、`head`、`tail`
- 能用 `find` 找到指定名称或类型的文件
- 能用 `grep` 从文本中筛出关键信息
- 能用 `wc`、`sort`、`uniq` 做基础统计整理
