# 01 Linux Basics

这一章是整套学习路线的起点，目标不是背很多命令，而是先把 Linux 的基本世界观建立起来。

学完这一章后，你应该能做到：

- 说清楚 Linux、发行版、内核、Shell、Terminal 分别是什么
- 看懂绝对路径、相对路径、当前目录、家目录这些概念
- 认识常见系统目录的大致用途
- 能读懂最基础的命令结构：命令 + 选项 + 参数
- 能独立使用 `pwd`、`ls`、`cd`、`whoami`、`man`、`--help`

## 建议学习顺序

1. 先看 `notes.md`
2. 再练 `commands.md`
3. 然后做 `labs/01-first-contact.md`
4. 最后用 `questions.md` 自测

## 文件说明

- `notes.md`: 核心概念和基础认知
- `commands.md`: 这一章最值得先上手的命令
- `questions.md`: 复盘题和参考答案
- `labs/01-first-contact.md`: 第一轮入门实验

## 学习建议

- 每读完一个概念，马上敲一次示例命令
- 不要求一次记住所有目录和术语，先形成大致印象
- 把你第一次看不懂的词记下来，过一遍实验后再回来看
- 如果你现在用的是 macOS，也可以先完成大部分基础练习

## 本章完成标准

如果下面这些你都能做到，说明这一章学得比较稳：

- 能解释 Shell 和 Terminal 的区别
- 能说清绝对路径和相对路径的区别
- 知道 `/home`、`/etc`、`/var`、`/tmp` 大概是干什么的
- 知道为什么输入命令后系统能找到对应程序
- 遇到陌生命令时，知道先用 `--help` 或 `man` 去看说明
