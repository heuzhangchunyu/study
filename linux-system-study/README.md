# Linux System Study

这是一个按学习路径整理的 Linux 系统学习目录。

建议学习顺序：

1. `00-roadmap`
2. `01-linux-basics`
3. `02-shell-and-commands`
4. `03-filesystem-and-text`
5. `04-users-and-permissions`
6. `05-processes-and-services`
7. `06-software-and-packages`
8. `07-networking`
9. `08-disk-and-storage`
10. `09-security`
11. `10-shell-scripting`
12. `11-practice-labs`
13. `12-cheatsheets`
14. `resources`

目录说明：

- `00-roadmap`: 学习路线、阶段目标、每周计划
- `01-linux-basics`: Linux 基础概念、发行版、目录结构、常见术语
- `02-shell-and-commands`: Shell 基础、常用命令、命令组合与管道
- `03-filesystem-and-text`: 文件系统、文件操作、文本查看与处理
- `04-users-and-permissions`: 用户、用户组、权限、sudo、chmod、chown
- `05-processes-and-services`: 进程管理、作业控制、systemd、服务管理
- `06-software-and-packages`: 软件安装、包管理、源码编译、环境变量
- `07-networking`: 网络基础、IP、端口、ssh、curl、wget、诊断命令
- `08-disk-and-storage`: 磁盘、分区、挂载、文件系统类型、容量排查
- `09-security`: 安全基础、防火墙、最小权限原则、日志审计
- `10-shell-scripting`: Bash 脚本、变量、条件、循环、函数、自动化
- `11-practice-labs`: 动手练习、实验记录、故障排查案例
- `12-cheatsheets`: 命令速查、常见操作模板、面试复习清单
- `resources`: 外部资料、参考链接、书籍、课程笔记

推荐你后续在每个主题目录里继续放这些内容：

- `notes.md`: 核心笔记
- `commands.md`: 常用命令与示例
- `labs/`: 实验练习
- `questions.md`: 易错点与复盘问题
