# 学习验收清单

## 基础认知

- [ ] 我知道 Linux、发行版、内核、Shell 分别是什么
- [ ] 我知道绝对路径和相对路径的区别
- [ ] 我能使用 `man` 或 `--help` 自己查命令

## 常用命令

- [ ] 我能独立完成目录切换和文件操作
- [ ] 我会使用重定向和管道组合命令
- [ ] 我能用 `grep` 和 `find` 查找内容或文件

## 文本处理

- [ ] 我会查看文件头部、尾部和分页内容
- [ ] 我会统计文本行数或关键词出现次数
- [ ] 我能从日志里过滤出关键信息

## 用户与权限

- [ ] 我理解用户、用户组、权限位的含义
- [ ] 我会使用 `chmod` 和 `chown`
- [ ] 我知道什么时候该用 `sudo`

## 进程与服务

- [ ] 我会查看正在运行的进程
- [ ] 我会结束异常进程
- [ ] 我知道服务和普通进程的区别

## 网络与存储

- [ ] 我会用 `ping`、`curl`、`ssh` 做基础排查
- [ ] 我会查看端口监听状态
- [ ] 我会用 `df` 和 `du` 检查磁盘空间

## 脚本与自动化

- [ ] 我会写最基础的 Bash 脚本
- [ ] 我会使用变量、条件和循环
- [ ] 我写过至少一个解决实际问题的小脚本

## 实战能力

- [ ] 我能独立完成一次基础环境巡检
- [ ] 我能写出一次问题排查过程
- [ ] 我整理过自己的 Linux 命令速查表

## 达标建议

如果大部分内容已经能勾选，说明你已经有了比较扎实的 Linux 入门能力。

如果有很多项还不能勾选，不用着急，优先回头补这三类：

1. 常用命令和路径概念
2. 权限、进程、网络这三块核心系统能力
3. 把零散命令串成完整操作流程的能力
