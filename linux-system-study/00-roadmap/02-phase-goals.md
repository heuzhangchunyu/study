# 阶段目标

## 第一阶段：入门基础

对应目录：

- `01-linux-basics`
- `02-shell-and-commands`
- `03-filesystem-and-text`

阶段目标：

- 了解 Linux 是什么、常见发行版和应用场景
- 熟悉终端、Shell、路径、命令格式
- 掌握文件和目录的常用操作
- 会使用文本查看和搜索命令

完成标志：

- 能熟练使用 `pwd`、`ls`、`cd`、`mkdir`、`cp`、`mv`、`rm`
- 能用 `cat`、`less`、`head`、`tail`、`grep` 查看和定位文本
- 能说清绝对路径和相对路径的区别

## 第二阶段：系统管理

对应目录：

- `04-users-and-permissions`
- `05-processes-and-services`
- `06-software-and-packages`

阶段目标：

- 理解用户、用户组、权限模型
- 掌握进程查看、终止、前后台作业控制
- 理解服务管理和软件安装方式
- 认识环境变量及其作用

完成标志：

- 能解释读写执行权限和 `chmod` 的常见写法
- 能使用 `ps`、`top`、`kill`、`jobs`、`bg`、`fg`
- 能使用一种常见包管理工具完成安装与查询

## 第三阶段：网络与存储

对应目录：

- `07-networking`
- `08-disk-and-storage`
- `09-security`

阶段目标：

- 理解 IP、端口、DNS、SSH 的基础概念
- 会做基本网络连通性排查
- 理解磁盘、分区、挂载和容量检查
- 建立基础安全意识

完成标志：

- 能使用 `ping`、`ss`、`netstat`、`curl`、`ssh` 做基础检查
- 能使用 `df`、`du`、`lsblk` 查看磁盘和空间占用
- 知道为什么不应随意使用 root

## 第四阶段：脚本与实战

对应目录：

- `10-shell-scripting`
- `11-practice-labs`
- `12-cheatsheets`

阶段目标：

- 能编写 Bash 基础脚本
- 会使用变量、条件、循环、函数和参数
- 能把零散命令组织成自动化脚本
- 能对典型问题做一轮完整排查

完成标志：

- 能写一个备份、日志清理或批量处理脚本
- 能根据报错信息定位问题的大致方向
- 能整理自己的 Linux 命令速查表
