7. [07-first-lab.md](/Users/zcy/Desktop/code/study/linux-system-study/00-roadmap/07-first-lab.md)

使用方式：

- 先看 `01-overview.md`，明确学习目标和学习方式
- 再看 `02-phase-goals.md`，知道每个阶段要掌握什么
- 按 `03-8-week-plan.md` 推进每周学习任务
- 用 `04-checklist.md` 做阶段验收和查漏补缺
- 学概念时同步阅读 `05-core-knowledge-explained.md`
- 学命令时直接照着 `06-command-examples.md` 动手练
- 第一轮学习结束后，用 `07-first-lab.md` 做完整练习

推荐节奏：

- 每周学习 4 到 6 天
- 每天 45 到 90 分钟
- 每个主题都做“看概念 + 跑命令 + 记笔记 + 做练习”

说明：

- `01` 到 `04` 偏学习导航和验收
- `05` 到 `07` 偏知识讲解和操作示例
- 本目录中的命令默认以 Linux 为主，`apt`、`systemctl`、`ss` 等命令请在真正的 Linux 环境中练习
- 像 `pwd`、`ls`、`cd`、`grep`、`find`、`chmod` 这类命令，在 macOS 终端里也能先练基础用法

学习成果建议沉淀到这些目录：

- `01-linux-basics`: 基础概念笔记
- `02-shell-and-commands`: 常用命令与练习
- `03-filesystem-and-text`: 文件系统和文本处理命令
- `04-users-and-permissions`: 权限和用户管理实验
- `05-processes-and-services`: 进程和服务排查记录
- `07-networking`: 网络诊断案例
- `10-shell-scripting`: 脚本练习
- `11-practice-labs`: 综合实验
